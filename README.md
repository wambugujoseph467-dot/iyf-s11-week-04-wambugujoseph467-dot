## Project Description
This project contains JavaScript practice exercises completed during Week 4. It covers JavaScript fundamentals including variables, data types, operators, functions, conditionals, switch statements, loops, and problem-solving challenges. The exercises are linked to an HTML page and the output is displayed in the browser console.

## Features
- JavaScript setup using `index.html` and `main.js`
- Variable declarations using `let` and `const`
- Data types using the `typeof` operator
- Arithmetic operators
- String operations and methods
- Comparison and logical operators
- Functions (declarations, expressions, and arrow functions)
- Default parameters
- If/Else statements
- Switch statements
- For loops
- While loops
- For...of loops
- Number printing challenge (1–100)

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/wambugujoseph467-dot/iyf-s11-week-04-wambugujoseph467-dot.git
```

2. Open the project folder in Visual Studio Code.

3. Open `index.html`.

4. Run the project using the **Live Server** extension.

5. Open your browser's Developer Tools (**F12**) and select the **Console** tab to view the JavaScript output.

## Challenges Faced
Some of the challenges I encountered included:
- Declaring the same variable (`count`) twice, which caused an "Identifier has already been declared" error.
- Understanding the difference between `main` and `master` branches when pushing to GitHub.
- Organizing multiple JavaScript exercises in one file without causing naming conflicts.
- Understanding how different functions takes place.
- Downloading and running the node.js.

These problems were solved by:
- Renaming duplicate variables.
- Using the correct Git branch.
- Testing each exercise separately before moving to the next one.
